01비전

ProjectFBDC는 자동공격 뱀써가 아니다. 매 슬래시가 플레이어의 결정이고, 매 보스 패턴이 읽고 대응해야 하는 시그널이며, 죽음마저 영구 성장으로 환원되는 — 30초 안에 손맛을 이해하고, 30분 후엔 빌드 다양성에 빠지는 게임.

세 기둥

⚔️
居合 · Iaido
차징 · 조준 · 결단

VS류의 자동공격이 아닌 이아이도(거합). LMB 홀드로 차징, 방향 조준, 릴리즈로 대시-슬래시. 한 번의 결단으로 다수를 베는 손맛.

🌸
仁王 · Nioh-style
텔레그래프 · 패리 · 회피

보스/엘리트는 색 시그널 → 패턴 → 응답. 노랑이면 패리, 빨강이면 회피. 잡몹은 VS처럼 그냥 베지만 보스는 읽어야 한다.

🏯
HD-2D
3D 월드 + 2D 빌보드

옥토패스 트래블러 스타일. 3D 그라운드와 카메라 깊이감 위에 2D 픽셀 스프라이트가 살아 움직이는 도트 비주얼.

플레이 루프

STEP 01
아웃게임 선택
챕터 선택 + 메타 패시브로 시작 능력치 보정 + 시작 카드 1장(언락 시)
STEP 02
인게임 생존
웨이브 처리 → 레벨업 → 3장 카드 → 빌드. 보스까지 ~2~5분
STEP 03
정산 / 사망
보스 격파 시 클리어 / 사망 시 그 시점까지 EXP·시간 기록. 둘 다 혼(魂) 적립
STEP 04
메타 성장
혼으로 패시브 강화 + 카드 풀 언락 → 다시 STEP 01. 매 회 더 강해진 상태로 도전

차별점 한 줄

VS: "자동공격 + 무한 패시브 스택" → 멍 때리는 손맛 · Soulslike: "패턴 학습 + 어려운 보스" → 죽음의 무게 · 본 프로젝트: 그 사이의 빈 공간 — 30분 단위로 끝나는 뱀써의 흡입력 위에, 보스마다 다른 시그널을 읽고 패리하는 인왕의 손맛을 얹는다.

02타이밍 액션 뱀써 — 핵심 정체성

"타이밍 액션 뱀써"는 본 프로젝트의 유일무이한 차별점이자 모든 시스템이 정렬해야 할 북극성이다. 이 섹션은 현재 살아있는 타이밍 메커닉을 정리하고, 정체성을 강화할 다음 단계 후보를 우선순위 순서로 나열한다.

DESIGN NORTH STAR

"매 입력이 결단이다 — 자동은 없다."

플레이어가 화면을 본다. 시그널을 읽는다. 차징·조준·릴리즈한다. 회피한다. 패리한다. 잘 했으면 보상받고 못 했으면 죽는다. 타이밍이 곧 게임플레이다.

장르 비교 — 우리는 어디에 있는가

장르 / 게임 자동공격 차징·조준 패리 완벽 회피 메타 진행
Vampire Survivors 완전 자동 없음 없음 없음 강함
Hades 없음 제한적 대시 무적만 대시 i-frame 강함
Sekiro 없음 강공격 핵심 메커닉 미카리 없음 (3D 액션 어드벤처)
Nioh 2 없음 강공격 + 스탠스 버스트 카운터 기 펄스 소울즈식
ProjectFBDC 의도적으로 제거 이아이도 차징 + 방향 조준 YELLOW 시그널 0.3s 윈도우 Shift 대시 i-frame 0.18s VS식 단순 메타

현재 살아있는 타이밍 메커닉

이아이도 차징 (Iaido Charge) Implemented
LMB 홀드 — 0초에서 0.72초까지 차징하면 사거리 3 → 11, 폭은 일정. 릴리즈 시 대시-슬래시. 대시 중 무적, 일반 적 즉사. 차징량 시각화는 AimArrow로. Player.gd / player_data.tres.
보스 노랑 패리 (Yellow Parry Window) Implemented
보스가 휘두르기 전 머리 위 색 시그널 + 지면 부채꼴. 노랑일 때 sweep 직전 0.3초 윈도우 안에 슬래시로 맞히면 패턴 캔슬 + 카메라 강쉐이크 + 보스 1초 스턴. 빨강일 땐 못 막음 — 회피 필수. Boss.gd.
대시 회피 i-frame Implemented
Shift — 0.18초 무적, 거리 3.5, 쿨 0.7초. 화살·잡몹 부채꼴 모두 통과. Wind Walker 카드로 쿨 -25% 스택.
엘리트 처치 불릿타임 (Type 3) Implemented
파란 엘리트 처치 시 3초간 모든 적 시간 25% 슬로우 + 환경 채도 깜빡임. 보상으로서의 시간 조작 — 플레이어가 만든 윈도우.

다음 단계 — 타이밍 액션 강화 제안

우선순위 순. 각 항목은 "이걸 넣었을 때 게임이 더 타이밍 게임으로 느껴지는가"를 기준으로 정렬됨.

⏱ 퍼펙트 패리 보상 사슬 (Perfect Parry Chain) Proposed · Milestone 2~3
보스 패리 성공 직후 1초 윈도우에 한해 다음 슬래시가 3중첩으로 들어감 (= 3뎀). 연속 패리 성공 시 카메라 줌인 + 음 한 박. 왜? 현재 패리 보상은 "보스 1초 스턴"뿐 — 카운터가 약하다. 보상을 키워야 노랑이 보일 때마다 욕망이 생긴다.
구현: Boss._on_parried에서 PC parry_boost_until 설정 → SlashAttack이 그 시간에 보스 hit 시 dmg×3.
⏱ 퍼펙트 닷지 → 카운터 윈도우 (Perfect Dodge → Counter) Proposed · Milestone 3
피격 직전 0.08초 안에 Shift를 쳤을 때 — i-frame 0.18s 유지 + 추가로 0.5초 자기-불릿타임 + 화면 가장자리 잔영 효과. 엘리트 3 효과의 PC 버전. 왜? 회피가 "안 맞으면 OK"의 패시브 행동에서 → "직전에 맞춰 치면 보상" 능동 행동으로 격상.
구현: Player._on_hit_attempt 시그널 도입 → 피격 판정 직전 dodge 이력 체크.
⏱ 차징 그레이드 (Iaido Grade — Quick / Mid / Perfect) Proposed · Milestone 3~4
현재 차징은 0~0.72s 선형. 이걸 3구간으로 나눔:
· Quick (0~0.3s) — 짧은 사거리·빠른 회복
· Mid (0.3~0.6s) — 균형
· Perfect (0.65~0.72s) — 최대 사거리 + 슬래시 trail에 보조 잔상 1개 (= 1초 안에 같은 자리에 한 번 더 친다)
0.72s를 넘기면 overcharge fizzle — 1초 모든 차징 봉인.
왜? 차징 길이의 "타이밍 결정"이 현재 단조롭다. 구간을 두면 음악적 리듬이 생긴다.
구현: Player.gd charge_time → grade 분기 + AimArrow에 grade band 색 추가.
⏱ 잡몹 텔레그래프 캔슬 (Mob Telegraph Cancel) Proposed · Milestone 3
근거리 잡몹 부채꼴 텔레그래프 (현재 0.5초) 중에 슬래시로 베면 텔레그래프 자체가 캔슬 + 잡몹은 즉사. 스킬샷의 보상 — "예방적 슬래시"가 의미를 갖는다. 왜? 현재 잡몹은 그냥 베면 다 죽으니 텔레그래프가 "맞으면 죽음" 외엔 의미가 없다.
구현: MeleeEnemy wind-up 중 take_hit 시 텔레그래프 효과 노드 destroy.
⏱ 젠 미터 (Zen Meter — 연속 퍼펙트 보상) Proposed · Milestone 4~5
퍼펙트 패리·퍼펙트 닷지·퍼펙트 차징이 연속될 때마다 Zen 미터 +1. 미터가 가득 차면 필살 슬래시 사용 가능 (대각선 풀화면 슬래시 1회). 피격 시 미터 절반 감소. 왜? "잘 했을 때만 더 잘할 수 있다"의 상승 사이클. Hi-Fi Rush의 리듬 미터, Devil May Cry의 SSS 랭크의 정신.
구현: ZenSystem.gd 매니저 신규. HUD 미터 표시. 필살은 SuperSlash.tscn 효과.
⏱ 다중 시그널 컬러 (Multi-Color Telegraph) Future · Milestone 6+
보스 2/3가 추가될 때 — 시그널 컬러 확장:
· 흰색 = 잡기 패턴 → Shift만 회피 가능 (패리 불가)
· 보라 = 광역 → 거리 벌리기만 가능
· 녹색 = 다단히트 → 패리 N회 연속 시에만 캔슬
각 보스는 컬러 조합이 달라야 한다. 보스1 = 노/빨, 보스2 = 노/빨/흰, 보스3 = 노/빨/보라/녹 — 점진 확장.
⏱ 비트 동기 모드 (Rhythm-Sync, 선택적) Future · Optional · Milestone 7+
BGM 도입 시점에 — 옵션으로 켜는 모드. 비트 박자 ±100ms 안에 슬래시 릴리즈 시 추가 데미지 +0.5 / Zen +1. Hi-Fi Rush 정신, 하지만 강제는 아니다. 왜 옵션? 음악에 약한 플레이어를 배제하면 안 됨. 접근성 위주.
⏱ 패리 카드 메커니즘 그룹 (Parry-Themed Cards) Future · Milestone 3
타이밍 정체성을 카드 풀에도 — 신규 카드 그룹:
· Counter Step — 패리 성공 후 1초 안에 이동 속도 +50%
· Echo Slash — 퍼펙트 차징 시 0.3초 후 같은 자리 CircularSlash
· Reflective Guard — i-frame 중 닿은 화살 반사
· Iron Will (Defiance) — 피격 시 자동으로 0.5초 i-frame + Zen +1
